# Proyecto Diplomado - IF

### 1. Subir el proyecto
``` bash
git clone https://github.com/Iris-Flores/diplomado.git
```
### 2. Instalar dependencias
``` bash
npm install
```
### 3. Copiar variables de entorno y configurar
``` bash
cp .env.sample .env
```
### 4. Levantar el proyecto
#### 4.1. En el desarrollo
``` bash
npm run dev
```
